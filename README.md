
# Chat Application

> Application for chatting 


## How to run:
- Clone this repo to your local machine using `https://github.com/Greedion/Chatting-Application.git`
- Run application
- Go to `localhost:8080`


## Technology
 - Spring Boot
 - STOMP Protocol
 - Embedded MessageBroker
 - WebSocket
 - Project Lombok
 - Javascript
 - HTML5
 - CSS
